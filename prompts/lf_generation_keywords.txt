### Response Format

List all new criteria in the response. For each, include:

- `criterion`: a short name for the criterion  
- `description`: a list of substrings separated by the "+" sign. Text is considered to belong to the class if it contains all the substrings in one text. Try not to use more than 2 short substrings (with length less than 10 characters, words can be stemmed) in one description and do not make descriptions too complex or too specific.
- `class`: the class to which the criterion applies. It ALWAYS must be {{ label_name }}

The description will be applied to the texts for classification using the python function below:

def _substring_LF(x, description, label):
    if "+" in description:
        description = description.split("+")
        return label if all([k in x.text.lower() for k in description]) else -1
    else:
        return label if description in x.text.lower() else -1

This function splits the description at the "+" symbol and checks whether each element is present in the classified string.
